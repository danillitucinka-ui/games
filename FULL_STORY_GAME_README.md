# Полная игра с картинками и сюжетом

## 🎨 Процедурно генерируемые спрайты

Игра включает систему генерации спрайтов, которая создает все визуальные элементы программно:

### Персонажи:
- **Koushi Shirota**: Синий цвет, спокойное выражение
- **Takane Takamine**: Красный цвет, цундере выражение с румянцем
- **Miki Sato**: Желтый цвет, веселое выражение
- **Seiya Ouji**: Синий цвет, загадочное выражение
- **Kiriko Shiratori**: Розовый цвет, элегантное выражение

### Фоны:
- **School Rooftop**: Небо с градиентом, здания, крыша
- **Classroom**: Стены, доска, парты
- **Garden**: Вишневые деревья, цветы
- **City**: Неоновые огни, здания
- **Warehouse**: Темные стены, ящики
- **Mountain**: Скалы, небо

### Эффекты:
- Ауры (светло-синие и розовые)
- Частицы (кровь, искры, пыль)
- Индикаторы здоровья
- Спецметры

## 📖 Полный сюжет "Please Put Them On, Takamine-san"

### Пролог:
- Знакомство с Коши Широта
- Его мысли о повседневной жизни
- Намек на предстоящие события

### Акт 1: Знакомство
- Первая встреча с Такане Такаминэ на крыше
- Её цундере реакция
- Непонимание и интерес

### Акт 2: Конфликт
- Знакомство с друзьями
- Развитие отношений
- Растущие чувства

### Акт 3: Разрешение
- Кульминация чувств
- Битвы и откровения
- Понимание друг друга

### Эпилог:
- Счастливый конец
- Намек на продолжение
- Цундере завершение

## 🎬 Кат-сцены

Каждый акт включает диалоговые сцены:
- Анимированные персонажи
- Диалоговые окна
- Переходы между сценами
- Музыкальное сопровождение

## 🎮 Режимы игры

### Story Mode:
- Полная история с кат-сценами
- 6 уровней с уникальными врагами
- Прогресс сохранения

### Другие режимы:
- Arcade: Бесконечные бои
- Survival: Волны врагов
- Versus: PvP с выбором персонажей
- Training: Безопасная практика

## 🖼️ Рекомендации по настоящим изображениям

Если хотите заменить процедурные спрайты на настоящие изображения:

### Где найти подходящие ассеты:
1. **OpenGameArt.org** - Бесплатные спрайты в стиле аниме
2. **Itch.io** - Ассеты персонажей и фоны
3. **Kenney.nl** - Качественные пиксель-арт ассеты
4. **Aseprite** - Создание собственных спрайтов

### Формат файлов:
- **PNG** с прозрачностью для персонажей
- **Размер**: 64x96 для персонажей, 1200x700 для фонов
- **Стиль**: Аниме, пиксель-арт

### Добавление в игру:
```cpp
// В SpriteGenerator добавить загрузку изображений
sf::Texture texture;
if (texture.loadFromFile("assets/characters/shirota.png")) {
    textures["shirota_real"] = texture;
}
```

1. Скачайте с GitHub Releases

3. Запустите `ShirotaBloodPolarity.exe`
