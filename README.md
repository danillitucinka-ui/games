# Shirota: Blood Polarity Ultimate

![Game Logo](https://via.placeholder.com/800x400/FF4444/FFFFFF?text=Shirota+Blood+Polarity)

A full-featured 2D fighting game based on the manga "Please Put Them On, Takamine-san", featuring beautiful anime-style graphics, multiple characters, levels, and game modes.

## 🎮 Features

### Game Modes
- **Story Mode**: Experience the full story with dialogues and cutscenes
- **Arcade Mode**: Classic endless fighting
- **Survival Mode**: Survive waves of enemies
- **Versus Mode**: Player vs Player battles
- **Training Mode**: Practice without consequences

### Characters
- **Koushi Shirota**: Light polarity strategist
- **Takane Takamine**: Dark polarity aggressor
- **Miki Sato**: Fast light attacker
- **Seiya Ouji**: Balanced neutral fighter
- **Kiriko Shiratori**: Elegant dark magic user

### Levels
- School Rooftop
- Classroom Chaos
- School Garden
- Night City
- Abandoned Warehouse
- Mountain Summit

### Advanced Features
- Combo system with scaling damage
- Special moves and polarity switching
- Achievement system
- Save/Load progress
- Multiple difficulty levels
- Enhanced audio and visual effects

## 🚀 Quick Start

### Download
- Visit the [Releases](https://github.com/yourusername/ShirotaBloodPolarity/releases) page
- Download the latest `ShirotaBloodPolarity-Windows.zip`
- Extract and run `ShirotaBloodPolarity.exe`

### Controls
- **Arrow Keys / WASD**: Move
- **Space**: Attack
- **Q/E**: Special attacks
- **Shift**: Change polarity
- **V**: Dash
- **S**: Block
- **ESC**: Pause

## 🛠️ Building from Source

### Prerequisites
- CMake 3.10+
- MinGW or Visual Studio
- SFML 2.5.1

### Build Steps
```bash
cd cpp_game
mkdir build && cd build
cmake .. -G "MinGW Makefiles"
mingw32-make
```

### Dependencies
- SFML Graphics, Window, System, Audio libraries
- Copy SFML DLLs to the executable directory
