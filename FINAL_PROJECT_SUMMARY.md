## 🚀 Как запустить:

### Автоматическая сборка (GitHub):
1. Push в репозиторий
2. GitHub Actions автоматически соберет .exe
3. Скачайте с Releases

### Ручная сборка:
```bash
cd cpp_game
mkdir build && cd build
cmake .. -G "MinGW Makefiles"
mingw32-make
cp *.dll .
./ShirotaBloodPolarity.exe
```
