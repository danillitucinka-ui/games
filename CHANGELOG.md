# Changelog

All notable changes to **Shirota: Blood Polarity Ultimate** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete game with 5 characters and 6 levels
- Multiple game modes (Story, Arcade, Survival, Versus, Training)
- Achievement system with 6 achievements
- Save/Load functionality
- Enhanced graphics with particle effects
- Programmatically generated audio system
- GitHub Actions CI/CD pipeline
- Comprehensive documentation

### Features
- Combo system with scaling damage
- Special moves and polarity switching
- Multiple difficulty levels
- Pause and settings menus
- Character selection with previews
- Level-specific environments and effects

## [1.0.0] - 2024-04-06

### Added
- Initial release of Shirota: Blood Polarity
- Basic fighting mechanics
- Two characters (Koushi Shirota, Takane Takamine)
- Single level gameplay
- Simple audio and graphics

### Known Issues
- Limited content
- No save system
- Basic AI
